## 📖 Sobre o Projeto

Este projeto reúne diversos exercícios e exemplos desenvolvidos em **Java**, simulando situações relacionadas a um sistema de teatro.

O objetivo principal é fortalecer conceitos fundamentais da programação, incluindo:

✨ Estruturas condicionais  
✨ Entrada e saída de dados  
✨ Tomada de decisões  
✨ Organização de código  
✨ Resolução de problemas

## ▶️ Como Executar

### Clone o repositório

```bash
git clone https://github.com/seu-usuario/Teatro.git
```

### Entre na pasta

```bash
cd Teatro
```

### Compile o exercício desejado

```bash
javac ex1.java
```

### Execute

```bash
java ex1
```
